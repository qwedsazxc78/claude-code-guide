# 第四章：Cowork — 不寫 code 也能用 AI

*不會寫 code？你也能用 AI 工作*

你有沒有遇過這種情況？

Downloads 資料夾裡堆了 2000 多個檔案，PDF、截圖、安裝檔、不知道哪來的 .zip——你知道應該整理，但每次打開就關掉。或者月底要報帳，桌上 40 張收據，光是打開每一張、手動輸入金額和日期，就花了一整個下午。

這些事不需要會寫程式。你需要的是一個能看懂你電腦裡檔案的 AI。

**Cowork 就是做這件事的。**

---

## 4.1 Cowork 是什麼？

Cowork 是 Claude Desktop App 裡的一個功能頁籤。你在上方看到三個 tab：**Chat / Cowork / Code**。

Chat 是純對話。Code 是寫程式。Cowork 在中間——它讓 Claude 直接存取你電腦上的檔案和資料夾，用自然語言告訴它你要做什麼，它就去做。

跟 ChatGPT 上傳檔案的差異：

| | ChatGPT 上傳 | Claude Cowork |
|--|-------------|---------------|
| 檔案數量 | 一次幾個 | 整個資料夾，幾千個都行 |
| 檔案大小 | 有上傳限制 | 讀本地檔案，沒有限制 |
| 操作能力 | 只能分析、回答 | 可以搬移、重命名、建立新檔案 |
| 工作位置 | 雲端 | 你的電腦上 |

重點來了：**Cowork 的核心能力是處理本地檔案。** 它不是雲端工具，它直接在你的電腦上操作。這意味著你的資料不需要上傳到任何地方，它就在你的硬碟上被處理。

### 怎麼開始

1. 打開 **Claude Desktop App**
2. 點上方的 **Cowork** tab
3. 選擇一個資料夾（「Work in a Folder」）
4. 用自然語言描述你要做什麼

就這樣。不需要安裝任何東西，不需要會用 Terminal。

### 方案需求

Cowork 需要 Claude Pro 以上的方案。如果你的任務比較複雜（處理大量檔案），建議用 Max 方案，額度比較充裕。

## 4.2 實戰案例：Downloads 資料夾大掃除

這是 Cowork 最經典的使用場景，也是讓很多人第一次說「我需要這個」的案例。

有人把 Cowork 指向一個有 **2200 個檔案**的 Downloads 資料夾。PDF、圖片、影片、安裝檔、各種 `file (1).pdf` 和 `Untitled (35).png`。

他只說了一句話：

```
幫我把 Downloads 裡的檔案整理好，
按照檔案內容分類到不同資料夾。
```

Cowork 不只看檔名——它讀了每個檔案的**實際內容**，然後建立了邏輯分類：

```
Downloads/
├── 社群媒體/        # 從各平台下載的圖片
├── AI 生成/         # AI 工具產出的圖片
├── Logo 與品牌/     # 公司 logo、brand assets
├── 個人照片/        # 手機匯出的照片
├── AI 研究/         # 論文、技術文章 PDF
├── 行銷素材/        # 簡報、宣傳品
├── 個人文件/        # 帳單、合約
├── 原型設計/        # Figma 匯出、wireframe
└── 備份/            # 舊版本、重複檔案
```

整個過程不到 20 分鐘。那些你拖了半年不想面對的 2200 個檔案，就這樣被分好了。

### 你可以試的第一步

不需要一次整理整個 Downloads。先試小的：

```
看一下 Downloads 裡面有什麼類型的檔案，
給我一個分類建議，先不要動任何檔案。
```

讓 Cowork 先掃描、先建議，你確認分類邏輯之後，再說「好，照這個分」。這跟第五章教的 Plan → Edit 工作流是一樣的思路——先看計畫，再動手。

### 注意事項

有一個真實案例要提醒你：有人跟 Cowork 說「幫我清理這個資料夾」，Cowork 把它認為是「雜物」的 11GB 檔案刪掉了。

所以：
- 說「**整理**」、「**分類**」，不要說「**清理**」
- 重要檔案先備份
- 讓 Cowork 先列出計畫，確認後再執行

## 4.3 實戰案例：收據照片 → Excel 記帳表

第二個案例更實用：把收據照片變成記帳表。

想像月底了，你有 40 張收據——有些是手機拍的照片，有些是 email 裡的 PDF。你要整理成一份報帳用的 Excel。

以前的做法：打開每張照片，手動輸入日期、商家、金額、付款方式。40 張大概要 2-3 小時。

用 Cowork 的做法：

```
把 receipts/ 資料夾裡的所有收據照片和 PDF 整理成 Excel，
每一筆要有：日期、商家名稱、金額、消費類別、付款方式。
最後加一個 summary sheet，按類別統計總金額。
```

Cowork 會：
1. 讀取每張照片和 PDF 的內容（它看得懂圖片上的文字）
2. 提取日期、商家、金額等資訊
3. 自動判斷消費類別（交通、餐飲、辦公用品...）
4. 產出一份格式化的 Excel，附公式和條件格式
5. 加上按類別的彙總統計

從 2-3 小時變成 5-10 分鐘。而且 Cowork 不會看錯金額——至少不會比你手動輸入時更容易看錯。

## 4.4 更多 Cowork 使用場景

Cowork 能做的事比你想的多。這裡列幾個真實案例：

**照片整理與重命名**
500 張旅遊照片從 IMG_4521.png 變成 beach-sunset-koh-lanta.jpg，自動按地點和主題分類。一個拖了兩年的整理工作，20 分鐘完成。
→ [MakeUseOf 實測](https://www.makeuseof.com/claude-cowork-renames-and-sorts-photos/)

**掃描文件自動改名**
一堆 scan_00247.pdf 變成 2026-01-15_Electric-Bill_$142.pdf。Cowork 讀取 PDF 內容，提取日期和描述，自動重命名。報稅前置作業的救星。
→ [MacSparky 示範](https://www.macsparky.com/blog/2026/01/file-renaming-with-claude-cowork/)

**2900 張截圖變成可搜尋檔案庫**
年積月累的「Screenshot 2024-03-15 at 10.42.31 AM.png」，Cowork 分析每張截圖內容，命名為 aws-lambda-error-dashboard 或 figma-homepage-redesign-v3。
→ [PandaiTech](https://pandaitech.my/alpha/organize-cluttered-screenshots-with-claude-cowork-5de68311)

**會議紀錄 → 行動項目**
丟進一週的會議逐字稿，Cowork 提取所有 action items、按負責人分組、產出追蹤表和 follow-up email 草稿。
→ [Cowork Guide](https://aiblewmymind.substack.com/p/claude-cowork-use-cases-guide)

**2 小時清完積壓工作**
一位使用者在 2 小時內讓 Cowork 產出：14 份拖了三個月的職缺描述、Q1 行銷策略文件、47 封一直沒回的 email、3 個網站頁面文案。
→ [Vibhu 的推文](https://x.com/vibhu/status/2011036579859427659)

## 4.5 Cowork vs Chat vs Code

回到那三個 tab：

| | Chat | Cowork | Code |
|--|------|--------|------|
| 做什麼 | 對話、問答、分析 | 處理本地檔案 | 寫程式、改 code |
| 誰用 | 所有人 | 所有人 | 工程師為主 |
| 改檔案嗎 | 不改 | 會搬移、重命名、建立 | 會修改程式碼 |
| 典型場景 | 「這是什麼意思？」 | 「幫我整理這些檔案」 | 「加一個 API endpoint」 |
| 風險程度 | 低 | 中（會動檔案） | 高（會改 code） |

**我的建議：** 如果你身邊有不寫 code 的朋友、同事、家人，花 10 分鐘帶他們看一次 Cowork。讓他們選一個自己電腦上「一直想整理但懶得動」的資料夾，體驗一次。他們會感謝你。

### Computer Use — 操作整台電腦

2026 年 3 月，Cowork 新增了 Computer Use（電腦使用）能力。Claude 可以直接控制你的 Mac（4 月擴展到 Windows）：點擊、輸入、切換 App、完成整個工作流。

這讓 Cowork 從「操作資料夾裡的檔案」升級到「操作整台電腦」。不過這個功能還在早期，目前的核心價值還是在本地檔案處理。

## Alex 用 Cowork 示範給客戶的那次

我有一個陪跑制客戶，USL 少年玩具的總經理，完全沒有技術背景。

我想讓他理解「AI 處理公司資料」這件事是什麼感覺，但打開 Terminal 只會讓他更困惑。所以我用 Cowork 示範。

我把他提供的一批倉儲進出貨 CSV 丟進資料夾，在 Claude Desktop App 裡選了那個資料夾，然後打了一段中文：「幫我分析這些 CSV，找出過去三個月哪些 SKU 的進出頻率最高，哪些有異常的庫存落差。」

Claude 讀了所有檔案，寫了一段分析腳本，跑起來，產出一份整理好的摘要。

整個過程不到 3 分鐘，沒有一行命令列，沒有打開任何 IDE。他看完說「這個我也可以用」——這才是重點。

Cowork 不是工程師的玩具，是讓 PM、設計師、業務、財務也能用 AI 能力的入口。
