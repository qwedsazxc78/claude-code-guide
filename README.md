# Claude Code 完全指南

> 從零開始，掌握 AI Coding 的完整工作流

這是一本免費的開源書，用繁體中文寫給每一個想認真學 Claude Code 的人。

不管你是工程師、產品人、還是對 AI Coding 好奇的非技術背景，這本書都從你實際會碰到的問題出發，帶你建立一套真正能用的 AI 開發工作流。

- **繁中首本開源**：目前市面上幾乎找不到完整的繁體中文 Claude Code 教學，這本書是第一本開源版，完全免費閱讀。

- **持續更新**：Claude Code 功能迭代快，這本書會跟著更新——你看到的永遠是當前可用的版本，不是半年前的截圖。

## 關於本書

這本書是「Claude Code 全攻略工作坊精華版」的文字版本，涵蓋四大支柱：

- **Chat**：如何跟 Claude 對話，精準傳達需求
- **Work**：如何讓 Claude 在你的專案裡動手工作
- **Config**：如何配置 CLAUDE.md、Hooks、Agents，讓 Claude 理解你的規範
- **Cloud Worker**：如何在背景執行任務、平行開發、自動化流程

書中所有範例都可以在真實專案裡操作。如果你想要完整的 Lab 環境和逐步練習，歡迎加入工作坊。
